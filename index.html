Usando addEventListener com Vários Botões

O addEventListener é uma forma moderna de adicionar ações a elementos HTML. Com ele, podemos fazer algo acontecer quando o usuário interage com a página, como clicar em um botão, passar o mouse ou digitar no teclado.

Abaixo estão alguns exemplos de como diferentes botões podem executar ações específicas ao serem clicados:

Texto interativo:

Esse texto será alterado pelos botões abaixo.

Trecho de código JavaScript:


document.getElementById("mudarCor").addEventListener("click", function() {
  document.getElementById("meuTexto").style.color = "rgb(255,255,255)";
});

document.getElementById("mudarTexto").addEventListener("click", function() {
  document.getElementById("meuTexto").textContent = "O texto foi alterado com sucesso!";
});

document.getElementById("esconderTexto").addEventListener("click", function() {
  document.getElementById("meuTexto").style.display = "none";
});

document.getElementById("mostrarTexto").addEventListener("click", function() {
  document.getElementById("meuTexto").style.display = "block";
});
